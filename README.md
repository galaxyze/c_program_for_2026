# c_program_for_2026
程设上机_嵌入式开发
Github使用：
1. 注册Github账号，新建一个仓库；
2. 将这个仓库 pull 到本地；
3. 在本地仓库中新建一个文件夹 ex01 ，创建 HelloWorld 程序；
4. 修改README.md 文件；
5. commit 第一个本地版本；
6. 增加程序功能， 使用 input 函数实现键盘输入，显示输出键盘输入内容；
7. 修改README.md 文件；
8. commit 第二个本地版本；
9. push 最终版本到 Github

* 步骤 3：已成功在 `ex01` 文件夹下创建 HelloWorld 程序。
